Improve Phase Components

 
Activity
–Develop, pilot, and implement solutions that address root causes
–Use data to evaluate both the solutions and the plans used to carry them out
•
 

Deliverables

–Planned, tested actions that eliminate or reduce the impact of the identified root causes
–Before and after data analysis that shows how much of the initial gap was closed
–A comparison of the plan to actual implementation

Tools Used in Improve Phase:

Brainstorming

Affinity Diagram

Control Charts

Run Charts

Value Stream Process Maps

FMEA

Communication Plan
